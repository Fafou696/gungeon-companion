INSTALLATION
1. Décompresse ce ZIP.
2. Dans GitHub, ouvre le dépôt gungeon-companion.
3. Add file > Upload files.
4. Sélectionne TOUS les fichiers du dossier décompressé.
5. Commit changes.
6. Attends la coche verte dans Actions / Deployments.
7. Ouvre :
   https://fafou696.github.io/gungeon-companion/
8. Sur iPhone, recharge la page.
9. Si v8.1 reste affiché, supprime l’ancienne icône de l’écran d’accueil,
   efface les données du site fafou696.github.io dans Safari, puis réinstalle l’app.

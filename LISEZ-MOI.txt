GUNGEON COMPANION — CORRECTIF V7.1

PROBLÈME CORRIGÉ
La v7 dépendait d’un dossier images contenant plus de 200 fichiers.
GitHub n’a probablement pas envoyé ce dossier correctement.

Dans la v7.1 :
- les 208 illustrations sont directement intégrées dans app.js
- aucun dossier images n’est nécessaire
- l’application fonctionne même si seuls les fichiers principaux sont envoyés
- version affichée : v7.1

INSTALLATION
1. Décompresse le ZIP.
2. Sur GitHub : Add file > Upload files.
3. Sélectionne tous les fichiers visibles et le dossier data.
4. Commit changes.
5. Attends 2 à 5 minutes.
6. Ouvre le lien dans Safari et actualise.
7. Ferme complètement l’application puis rouvre-la.
8. Vérifie que la version affiche v7.1.

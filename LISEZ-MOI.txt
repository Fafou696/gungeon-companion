GUNGEON COMPANION — MISE À JOUR V2

Cette version contient 167 fiches et ajoute :
- beaucoup plus d’armes, objets, boss, monstres, PNJ et étages
- export de progression
- import de progression
- cache v2 pour forcer la mise à jour de l’application installée
- tri alphabétique

MISE À JOUR SUR GITHUB
1. Décompresse cette archive.
2. Dans ton dépôt GitHub, clique Add file > Upload files.
3. Glisse TOUS les fichiers de ce dossier.
4. GitHub indiquera que les fichiers existants seront remplacés.
5. Clique Commit changes.
6. Attends environ 1 à 3 minutes.

SUR L’IPHONE
1. Ferme complètement Gungeon Companion.
2. Ouvre d’abord le lien dans Safari :
   https://fafou696.github.io/gungeon-companion/
3. Actualise la page.
4. Ferme Safari puis rouvre l’application depuis son icône.
5. La ligne de progression doit afficher v2.0.

INSTALLATION
1. Décompresse le ZIP.
2. Sur GitHub : Add file > Upload files.
3. Sélectionne tous les fichiers.
4. Commit changes.
5. Attends 1 à 3 minutes.
6. Ouvre le lien dans Safari et actualise.
7. Ferme complètement l’application installée.
8. Rouvre l’application.

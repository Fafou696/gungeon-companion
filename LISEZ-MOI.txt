GUNGEON COMPANION — V9.0

NOUVEAUTÉ PRINCIPALE
Les deux grandes planches ont été découpées automatiquement en pictogrammes
individuels sélectionnables en un seul toucher.

PICTOGRAMMES EXTRAITS
- Armes : 43
- Objets : 6
- Total : 49

UTILISATION
1. Ouvre une fiche.
2. Touche « Choisir le pictogramme de cette fiche ».
3. Choisis Armes ou Objets.
4. Touche directement le dessin.
5. Le choix est sauvegardé automatiquement.

INSTALLATION
1. Décompresse le ZIP.
2. GitHub > Add file > Upload files.
3. Envoie tous les fichiers.
4. Commit changes.
5. Attends le déploiement GitHub Pages.
6. Ouvre https://fafou696.github.io/gungeon-companion/
7. Vérifie que la version affiche v9.0.

IMPORTANT
Le découpage est automatique à partir des images fournies. Quelques vignettes
peuvent contenir une petite partie du fond ou deux dessins très proches.

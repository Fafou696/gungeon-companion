INSTALLATION
1. Décompresse le ZIP.
2. Dans GitHub : Add file > Upload files.
3. Sélectionne TOUS les fichiers ET les dossiers data et images.
4. Commit changes.
5. Attends 2 à 5 minutes, car il y a plus de 200 images.
6. Ouvre le lien dans Safari et actualise.
7. Ferme complètement l’application puis rouvre-la.
8. Vérifie que la version indique v7.0.
